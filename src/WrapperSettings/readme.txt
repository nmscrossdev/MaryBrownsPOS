localstorage saved values by using these keys:-

PRINTER SETTINGS----

selected_printer - Which printer do you want to use?
selected_papersize - Which paper size?
is_autoprint_receipt - Auto print receipt after each sale?

FRONT FACING DISPLAY-----
selected_ffdisplay - Display On
selected_wtdisplay - What do you want to display?
Which App do you want to show?

CASH DRAWER------
selected_drawer - Open cash drawer after each sale?
